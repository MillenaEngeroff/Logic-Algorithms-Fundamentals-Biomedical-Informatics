# Logic of Programming & Algorithms | Biomedical Informatics 🧬💻

This repository contains exercises and algorithms developed in **Portugol (VisuAlg 3.0)**. It represents the foundational logic and computational reasoning being built during my **Biomedical Informatics** degree.

### 🎯 Objective
Developing solid algorithmic thinking is crucial for building reliable health-tech solutions. These exercises focus on:
* **Conditional Structures** (Control flow for clinical decision support)
* **Loops** (Data iteration and processing)
* **Vectors & Matrices** (Initial logic for image and signal processing)

**Tools:** VisuAlg 3.0  
**Focus:** Logic, Problem Solving, Healthcare Algorithms.
